# Projeto Catálogo - BlueTech

## Descrição do projeto: 
Catálogo de livros onde é possivel adicionar livros já lidos, editar e deletar, além de conferir os detalhes. Todo o projeto foi realiado com JavaScript, HTML, CSS, Express e banco de dados PostgreeSQL, utilizando o deploy pelo Heroku.
